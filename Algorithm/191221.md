# 알고리즘 - 순열과 조합에 대한 이해

1. 순열
> **nPr = (n!) / (n-r)! **

```
// 순열의 개수 구하기
int cal_nPr(int a, int b){
    int n = 1, r = 1;
    for(int i=1; i<=a; i++)
        n * = i;
    for(int i=1; i<=(a-b); i++)
        r * = i;
    return n / r;
}
```

2. 조합
> **nCr = ( n! ) / r! * (n-r)!**

```
// 조합의 개수 구하기
int cal_nCr(int a, int b){
    int n = 1, r = 1, s = 1;
    for(int i=1; i<=a; i++)
        n * = i;
    for(int i=1; i<=b; i++)
        r * = i;
    for(int i=1; i<=(a-b); i++)
        s * = i;
    return n / r * s;
}
```

 > Point 2. Group 판단 후 얻은 정보들이 꽤나 많고 sort 해야 하는데, 이를 어떻게 쉽게 할지?

 - 임의의 pair를 정의한 후 max값을 적용하는 방식이 깔끔하다.

 ```
 pair<pii, pii> mx = {{-1, -1}, {-1, -1}};

for(){
    for(){
         pair<pii, pii> ret = bfs(x, y);
         mx = max(mx, ret); // 갱신
    }
}

if(mx.first.first < 2) break;

```

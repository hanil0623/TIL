## 알고리즘 - 에라토스테네스의 체

- 잘못 알고 있었던 부분 수정

```
bool a[1000005];
int main() {
	cin >> m >> n;
	a[1] = true;
	for (ll i = 2; i <= n; i++) {
		if (a[i]) continue;
		for (ll j = i * i; j <= n; j += i) {
			if (a[j]) continue;
			a[j] = true;
		}
	}
	
	for (int i = m; i <= n; i++) {
		if (a[i] == false) cout << i << '\n';
	}

	return 0;
}
```

- 0과 1은 소수가 아니니 true 처리하며, false인 부분만 소수이다.
